# assistente-voz-whisper-chatpgt

Projeto de assistente de voz com IA usando Whisper, ChatGPT e gTTS. 

## 📌 Descrição

Projeto que permite conversar por voz com uma IA. O áudio é capturado, convertido em texto, enviado para o modelo de linguagem e a resposta é retornada em áudio.

## 🚀 Tecnologias

- Python
- Whisper (Speech-to-Text)
- OpenAI (ChatGPT)
- gTTS (Text-to-Speech)
- Google Colab

## 🔄 Fluxo da Aplicação

- Usuário fala no microfone
- Áudio é convertido em texto
- Texto é enviado para IA
- IA gera resposta
- Resposta é convertida em voz

## ▶️ Como executar

- Abrir no Google Colab
- Inserir a API Key da OpenAI
- Executar as células
- Permitir acesso ao microfone

## ✨ Melhorias realizadas

- Adição de mensagens de status no fluxo
- Personalização da resposta da IA
